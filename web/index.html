jdawson.au

Source Code
This site in itself is completely coded by hand to allow complete control of the design. The idea behind this design is to create a personal relationship with the viewer by having the full and complete focus on the text, which is also enhanced by the fact that to actually interact with the site they need to place their mouse within this text, and almost physically moving themselves closer to me. It was also built with scalability in mind so it would be easy to place future projects onto this site.
